## 🌟 Features

### Customer Experience
- **Intuitive Shopping Cart**: Browse a diverse food menu and seamlessly add or remove items from the cart.

- **Promotion System**: Integrated coupon functionality allowing users to apply promo codes for discounts.

- **Order Tracking**: A dedicated tracking page to monitor order status from kitchen to delivery.

- **Secure Checkout**: Fully integrated with Stripe for safe and reliable credit card transactions.

### Admin Dashboard (Back-office)
- **Menu Management**: Complete CRUD functionality (Create, Read, Update, Delete) for food items and categories.

- **Promotion Management**: Full control over marketing campaigns with the ability to manage promo codes and discount rates.

- **Order Management**: Centralized dashboard to view and update the status of incoming orders.

## 🛠️ Tech Stack

### Frontend

- **React**: For building a dynamic and responsive user interface.

- **CSS**: Custom styling to ensure a clean, user-friendly, and mobile-responsive layout.

### Backend
- **Node.js & Express**: A robust server environment to handle API requests and business logic.

- **MongoDB**: A NoSQL database used for flexible data modeling of products, users, and orders.

### Payment & Integration
- **Stripe API**: Industry-standard payment processing for secure online transactions.
